# Github-User-Activity
A simple command-line interface activity displayer implemented in Python.

## Description
This project allows users to fetch the recent activity of a GitHub user and display it in the terminal.

## Installation:
1. Clone the repository:
```bash
git clone https://github.com/eezzatt/Github-User-Activity.git
```

2. Navigate into the folder:
```bash
cd GitHub-User-Activity
```

## Execution:
Run:
```bash
python activity-tracker.py (username)
```
